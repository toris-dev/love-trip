# 📝 회의록

이 디렉토리는 프로젝트 관련 회의록을 보관합니다.

## 📋 회의록 작성 가이드

### 파일 명명 규칙

```
YYYY-MM-DD-meeting-topic.md
```

예시:

- `2024-01-15-architecture-review.md`
- `2024-01-20-sprint-planning.md`

### 회의록 템플릿

```markdown
# 회의 제목

**날짜**: YYYY-MM-DD
**참석자**: 이름1, 이름2, 이름3
**회의 시간**: HH:MM - HH:MM

## 📋 안건

1. 안건 1
2. 안건 2
3. 안건 3

## 💬 논의 사항

### 안건 1

- 논의 내용
- 결정 사항

### 안건 2

- 논의 내용
- 결정 사항

## ✅ 액션 아이템

- [ ] 작업 1 (담당자: 이름, 마감일: YYYY-MM-DD)
- [ ] 작업 2 (담당자: 이름, 마감일: YYYY-MM-DD)

## 📝 다음 회의

- 날짜: YYYY-MM-DD
- 안건: 예정된 안건들
```

## 📚 회의록 목록

회의록은 날짜순으로 정리됩니다.

---

**참고**: 회의록은 GitHub에 커밋하여 모든 팀원이 접근할 수 있도록 합니다.
