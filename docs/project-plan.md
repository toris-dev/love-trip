## 1. 서비스 개요

**한 줄 설명:**

연인의 여행을 위해 교통편 · 숙소 · 데이트 장소 · 경비까지 한 번에 추천해주는 **커플 맞춤형 여행 설계 서비스**

**핵심 가치**

- 연인이 함께 계획할 수 있는 **공동 여행 설계 툴**
- **사용자들이 직접 만든 데이트/여행 코스 탐색 및 제작**
- 여행 후 **추억·지출 기록까지 관리되는 올인원 플랫폼**
- **사용자 생성 코스 공유 커뮤니티**로 지속적인 콘텐츠 확장

---

## 2. 타깃 고객

- 20~30대 커플 (국내 1박 2일 ~ 3박 4일 여행)
- 장거리 연애 중 정기 여행 계획 커플
- 생일/기념일/프로포즈 등 특별한 날 이벤트 여행 준비 커플
- 여행 코스를 공유하고 탐색하고 싶은 커플

---

## 3. 핵심 문제 & 해결 전략

| 커플 여행의 문제                             | 서비스 솔루션                                                   |
| -------------------------------------------- | --------------------------------------------------------------- |
| 경비 계산 & 분담이 번거롭고 불공평 느낌 발생 | 1/N 정산 / 지출 기록 / 예산 초과 경고                           |
| 교통, 숙소, 데이트 장소를 따로 검색해야 함   | AI 기반 **여행 + 데이트 코스 패키지 추천**                      |
| 기념일/서프라이즈/분위기 맞추기가 어려움     | **테마 코스 & 이벤트/서프라이즈 제안 기능**                     |
| 검증된 커플 여행 코스 정보 부족              | **사용자 생성 코스 공유 커뮤니티** + 실제 다녀온 후기 기반 추천 |
| 여행 계획에 대한 동기 부족                   | **게이미피케이션 보상 시스템** (XP, 포인트, 배지)               |

## 4. 주요 기능

### ① UGC 데이트 코스 탐색 & 여행 계획 제작

- 다른 사용자들이 만든 데이트 코스와 여행 코스 탐색
- 나만의 여행 계획 제작: 출발지/예산/분위기/스케줄 입력 → 교통 + 숙소 + 데이트 루트 구성
- 사용자들이 공유한 데이트 코스 및 여행 코스 수백 개 (지역별, 테마별)

**우선순위**: 🔴 High

**완료 조건**:

- 카페, 레스토랑, 야경 명소, 드라이브 코스, 전시/액티비티 추천

## 5. 구독 모델 설계 (핵심)

| 구분          | 미구독자 (무료)           | 프리미엄 구독자                             |
| ------------- | ------------------------- | ------------------------------------------- |
| 여행 추천     | 기본 코스 / 숙소 / 맛집   | **테마/기념일/럭셔리/서프라이즈 코스 포함** |
| 장소 정보     | 일반 리뷰 & 추천          | **커플 감성 큐레이션 필터**                 |
| 일정 변경     | 날씨/교통 알림만          | **일정 자동 재편성**                        |
| 예산 관리     | 정산/지출 입력 가능       | **예산 최적화 대안 코스 추천**              |
| 캘린더        | 일정 공유                 | **기념일 알림 + 예약 리마인더**             |
| 추억 저장     | 사진/기록 제한            | **앨범 무제한 + 커플 히스토리 리포트**      |
| **코스 공개** | **공개 가능 (기본 보상)** | **프리미엄 배지 + 우선 노출**               |
| **코스 탐색** | **기본 필터링**           | **고급 필터 + 맞춤 추천**                   |
| 혜택          | 광고 포함                 | **Ad-Free**                                 |

**구독 핵심 가치:**

> 단순 여행 계획이 아니라, 연인을 위한 "완성도 있는 감동 설계" 제공

**우선순위**: 🟢 Low (Phase 2)

**완료 조건**:

- [x] 구독 시스템
- [ ] 결제 연동

## 6. 수익 모델 (BM)

1. **예약 제휴 수익 (숙소/교통/레스토랑/액티비티)**
   - 사용자가 공개한 코스 내 장소 예약 연동
2. **프리미엄 구독**
   - 기념일 코스, 이벤트 추천, 예약 리마인더, 감성 필터
   - 제휴 할인 혜택
   - 프리미엄 배지 및 코스 우선 노출
3. **커플 선물/이벤트 제휴**
   - 꽃배달, 케이크, 선물샵, 이벤트 플래너 연동
   - 인기 코스 작성자에게 제휴 혜택 제공
4. **브랜드 협찬 (장기)**
   - 여행지/숙소/레스토랑 콜라보 코스 제작
   - 인기 코스 작성자와 브랜드 협업 프로그램
5. **UGC 기반 콘텐츠 확장 (신규)**
   - 사용자 생성 코스로 데이터베이스 자동 확장
   - 검증된 인기 코스의 프리미엄 배치 및 추천
   - 커뮤니티 활성화로 SEO 및 자연 유입 증가

## 8. 기대 효과

- 여행 준비 스트레스 감소 → **사용자 경험 향상**
- "커플 여행 전문"이라는 **시장 내 차별화 포지션 확보**
- 커플 특화 소비 시장(맛집/기념일/선물)과의 **교차 수익 창출**
- **UGC 기반 자동 콘텐츠 확장**으로 운영 비용 절감 및 지속적 성장
- **게이미피케이션**으로 사용자 참여 및 리텐션 향상
- **커뮤니티 형성**으로 자연스러운 바이럴 및 유기적 성장

## 9. MVP 출시 범위 제안 (우선순위)

### Phase 1: 핵심 기능 (2-3주)

1. **UGC 데이트 코스 탐색 & 여행 계획 제작**
   - 사용자들이 만든 코스 수백 개 탐색 (데이트 코스 + 여행 코스)
   - 코스 선택 및 나만의 여행 계획 제작 및 저장 기능
2. **커플 테마 장소 큐레이션**
   - 지역별, 테마별 장소 추천
   - 장소 상세 정보 및 지도 연동
3. **예산 추정 및 관리**
   - 장소별 `price_level` 기반 예산 자동 추정
   - 예산 항목별 입력 및 관리

### Phase 2: 커뮤니티 기능 (1-2주)

1. **사용자 코스 공개 기능**
   - 여행 계획 저장 시 "공개하기" 옵션
   - 공개 코스 탐색 페이지 (`/courses`)
   - 좋아요/저장 기능
2. **게이미피케이션 보상 시스템**
   - 코스 공개 시 XP/포인트 지급
   - 첫 코스 공개 시 배지 지급
   - 보상 알림 UI

### Phase 3: 고도화 기능 (1-2주)

1. **지출 기록 & 1/N 정산**
   - 여행 중 지출 입력
   - 커플 간 자동 정산
2. **프리미엄 기능: 기념일 테마 코스 + 기념일 알림 & 예약 리마인더**
   - 기념일 코스 추천
   - 알림 시스템

→ **최소 기능으로도 감성적 가치 + 구독 유도 효과 + 커뮤니티 활성화 극대화**

## 10. 기술적 구현 전략 (신규)

### 데이터베이스 설계

- **초기 코스**: `travel_courses`, `date_courses` 테이블 (관리자 생성)
- **사용자 생성 코스**: `user_courses`, `user_course_places` 테이블
- **상호작용**: `user_course_likes`, `user_course_saves` 테이블
- **보상 시스템**: `course_rewards` 테이블 (기존 게이미피케이션 시스템 연동)

### API 구조

- 코스 공개: `POST /api/user-courses/[id]/publish`
- 공개 코스 탐색: `GET /api/user-courses/public`
- 좋아요/저장: `POST /api/user-courses/[id]/like`, `/save`
- 보상 지급: `POST /api/rewards/grant`

### 보상 규칙

- 코스 공개: XP 100, 포인트 50, 배지 (첫 코스)
- 좋아요 받기: XP 5, 포인트 2 (일일 한도 100회)
- 저장 받기: XP 10, 포인트 5 (일일 한도 100회)
- 조회수 보상: 10회(20 XP), 50회(50 XP), 100회(100 XP), 500회(200 XP)
- 주간 TOP 10: 1위(500 XP), 2-10위(200 XP)

1. 사용자가 앱에 접속
2. "여행 계획하기" 클릭
3. 출발지(서울), 목적지(부산), 날짜(2박3일), 예산(50만원) 입력
4. 시스템이 최적 코스 추천:
   - 교통편: KTX 왕복
   - 숙소: 해운대 근처 호텔
   - 데이트 코스: 해운대 해수욕장 → 감천문화마을 → 자갈치시장
5. 사용자가 코스 확인 및 수정
6. 예산 확인 및 저장

### 시나리오 2: 당일 데이트 코스 찾기

1. 사용자가 "데이트 코스" 메뉴 클릭
2. 지역 선택 (강남)
3. 테마 선택 (로맨틱)
4. 추천 코스 목록 확인
5. 코스 상세 정보 확인 (장소, 예상 비용, 소요 시간)
6. 코스 저장 또는 공유

### 시나리오 3: 커플과 여행 계획 공유

1. 사용자가 여행 계획 생성
2. "커플과 공유" 클릭
3. 파트너에게 초대 링크 전송
4. 파트너가 계획 확인 및 수정
5. 실시간으로 계획 동기화
6. 여행 후 지출 기록 및 정산

### 시나리오 4: 코스 공개 및 커뮤니티 참여

1. 사용자가 여행 계획 완성
2. "코스 공개하기" 클릭
3. 코스 제목, 설명, 사진 추가
4. 공개 후 즉시 보상 수령 (XP 100, 포인트 50, 배지)
5. 다른 사용자들이 코스 탐색 및 좋아요/저장
6. 조회수 증가에 따른 추가 보상 수령

## 🛠 기술 요구사항

### 필수 기술

- **Frontend**: Next.js 15+, React 19+, TypeScript
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **외부 API**:
  - Tour API (한국관광공사)
  - 네이버 지도 API
- **배포**: Vercel (예정)

### 선택 기술

- **푸시 알림**: Web Push API
- **결제**: 토스페이먼츠 (구독 모델용)
- **OCR**: Tesseract.js 또는 클라우드 OCR 서비스
